UNDERSTAND

Loops are tools which allow us to repeat the actions of a piece of code without writing code additional times.

LEARN

For Loops

For loops are loops which repeat until a specified condition is evaluated as false. The elements of a For loop can be seen in the code block below.

for ([initialExpression]; [condition]; [incrementExpression])
 statement

Below is an example of a For Loop which prints the term 'This is loop number i' the number of times specified in the condition, where i indicates what loop has occurred. As the condition is set to 'i is less than 5', this code will loop 5 times.

var text = ""
var i;
for (i = 0; i < 5; i++) {   //Dictates the code is repeated 5 times
 text += "This is loop number " + i + "
"; }

While Loops

While loops are loops which repeat continuously while a condition is true. The elements of a While loop can be seen in the code block below.

while (condition) {
 Executable Code}

Below is an example of a While Loop which loops the number of times specified in the condition. In this condition, the while loop repeats continuously while it is true that i is less than 5. Each Time the code reaches the section designated i++, i increases by one, with the new value for i being stored outside the loop. It is then used for the next iteration of the loop until i < 5 is no longer true. As you can see, the while loop is defined rather differently to the For loop, with no increment value or initial value entered into the while condition.

var i = 0 while (i < 5) { i++; }
