UNDERSTAND

Developers use conditions in JavaScript when a decision is required in the code. We make decisions all the time in our day to day lives. An example of this could be something like the following: If it is a sunny day today I will go to the beach. To write decisions in code we use if, else and else if statements.

LEARN

If Statement

The if statement is used to evaluate whether a condition is true or false. If it is true then the following code block will be executed, if it is false it will not be executed. Using our example from earlier we can turn our personal decision into a code block.

if(weather == sunny) {
 go to the beach
}

With this example if the weather is not sunny nothing will happen but if it is sunny we will go to the beach. Try and think of some other decisions you have made recently and see if you can write them out in code versions. We can also combine multiple conditions in one if statement by using && / || operators. These require both or either condition to be true respectively for the code to run. The following is an example of the && operator.

if(weather == sunny && windSpeed != windy) {
 go to the beach
}

This will only run if the weather is sunny and it is also not windy.

Else Statements

Sometimes multiple outcomes to a decision are required. By using the else statement we can tell the program that when the if statement's condition is not met, do another option. Going back to our earlier example we could say that if the weather is not sunny, instead we will go to the movies. Lets take a look at how this would appear in a code block.

if(weather == sunny) {
 go to the beach
} else {
  go to the movies
}

Else If

Sometimes we want to be able to evaluate multiple conditions and run different code blocks depending on which is true. To do this we can use the else if statement. This means that if the previous if statement is false it will evaluate another condition and if that is true run a different block. Let us take a look at an example.

if (weather == sunny) {
  go to the beach
} else if(weather == windy) {
 go fly kites
} else {
 go to the movies
}

This code block will result in going to the beach on a sunny day, flying kites on a windy day, and going to the movies any other time. We can add as many else if statements as needed as well as combining as many and/or operators as we want.
