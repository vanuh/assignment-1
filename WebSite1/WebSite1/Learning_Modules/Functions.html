UNDERSTAND

In this section we will look at using functions in our code. JavaScript functions are blocks of code designed to perform a particular task. We can 'call' the function to cause it to run. Functions are incredibly useful as we can use them multiple times with different parameters to produce different results.

LEARN

Writing Functions

When we write functions we have the function keyword, followed by our function name, followed by parantheses (). The function name can contain letters, digits, underscores and dollar signs just like variables. Inside the parantheses we can include parameter names separated by commas. These parameters work as local variables within the function. Functions do not require any parameters to be set. Lets look at an example:

function product_of_numbers(n1, n2) {
  return n1 * n2   // The function returns the product of n1 and n2
}

var x = product_of_numbers(5, 10)   // value of x will be the result of the function which is 50

Local Variables

When declaring variables inside of a function they become LOCAL to the JS function. This means that they cannot be accessed from outside the function. This means that we can use variables with the same name inside different functinos. It also means that if we try to use the variable from within the function in a different piece of code it will not work. The code block below illustrates this.

// code here cannot use myVar

function myFunction() {
  var myVar = "I am a variable";
  // code written here can access myVar
}

// code written here cannot use myVar

function myFunction() {
  var myVar = "I am a different variable from the previous 'myVar'";
  // code written here can access myVar of this function
}
