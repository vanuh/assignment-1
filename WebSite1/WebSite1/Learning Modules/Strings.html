LEARN

Strings

The use of strings is very important as they allow us to store textual data instead of just using numbers. Javascript strings are defined using quotes to surround the text. It is important to note that both single and double quotation marks can be used as long as the same one is used to start and end the string. This comes in handy as it lets us use quotation marks within the string as well. Lets look at some examples.

var doubleQuoteString = "I am a string using double quotation marks"

var singleQuoteString = 'I am a string using single quotation marks'

var quoteString = "This 'string' uses single quotes inside it to emphasise the word string"

Escape Characters

Because strings must be enclosed within quotes some strings will be misunderstood by javascript. For example: var x = "They called him "tough" but really he was terrified" would be chopped to "They called him" because of the double quotes inside the string. To avoid this we can escape characters by using the backslash. \' will result in ', \" will result in " and \\ will result in \. Lets take a look at some more examples:

var x = 'It\'s okay.' // It's okay.

var y = 'It's okay.' // It

String Methods

String Length

Here we will look at some basic methods we can use on strings to return information about them. The first one we will look at is string length. This property returns the length of a string which tells us how many characters are in a string. Lets look at a quick example.

var str = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";

var stringLength = str.length; // Returns 26 as there are 26 letters in the string of the alphabet.

Slice String

The next method we will look at is the slice method. This allows us to specify a starting index and the ending index and take out a portion of the string between those positions. If you do not specify a second parameter it will slice from the starting index to the end of the string. It is important to note that the first character of a string is at index 0. Another point to note is that it will not include the last index. Lets look at some examples:

var str = "Frog, Lemur, Giraffe";

var strLemur = str.slice(6,11); // Value will be Lemur

var strFrog = str.slice(0,4); // Value will be frog. Note that it does not include the character at index 4 which would be a comma.

var strLemurGiraffe = str.slice(6); // Value will be Lemur, Giraffe

Replace Method

The last method we will look at in this lesson is the replace() method. This lets us replace a specified value in a string with a new value. By default the method will only replace the first match and is case sensitive however there are ways to manipulate that. Let us look at some examples:

var str = "Welcome to TerribleJavaScript! Learning made easy!";

var betterStr = str.replace("TerribleJavaScript", "JavaScript 1,2,3);

This will replace the word TerribleJavaScript with JavaScript 1,2,3. But what if our string was longer and had the competitions name in it multiple times such as "TerribleJavaScript is the best place to learn JS! Come to TerribleJavaScript for all your JS needs!". In its current form the method would only replace the first instance of the word. Luckily for us we can use a global match flag /g. That looks like the following:

var str = "TerribleJavaScript is the best place to learn JS! Come to TerribleJavaScript for all your JS needs!";

var betterStr = str.replace(/TerribleJavaScript/g, "JavaScript 1,2,3);

This would replace all the instances of our competitors name! Note the absense of quotes when using the /g flag.
