UNDERSTAND

Variables are containers in which we store data values. Variables allow programmers increased flexibility, as rather than entering a constant value, variables are able to change based on actions performed upon them when the program is excuted. This allows for a single program to work with different sets of data. Each variable has a name and a data type. Data types for variables include integers, floating point numbers and strings.

LEARN

In Javascript, the term 'var' followed by a space and a variable name (which could be anything) defines a variable. To fill the variable with content, you must place an ' = ' after the variable name and then input the desired text. If it is a word, quotation marks around the text must be entered to store it correctly. Numbers may be entered without quotation marks.


Examples of Variables:

var bignumber = 20000

var favcolour = 'Blue';

var wizard = 'Merlin';

var answer = 'Yes';
