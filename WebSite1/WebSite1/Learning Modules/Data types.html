UNDERSTAND

Following on from the lesson on variables we will look at the some different data types that can be held in variables. These can include numbers, strings, and booleans. Strings must be entered within quotation marks while numbers can be entered on their own. Boolean data types return a true or false value. Variables can contain one type of data and later contain another type of data depending on what is entered

LEARN

Numbers

The number data type can be used to input numbers with or without decimals. Different mathematical operations can then be used on the numbers to manipulate them further. Numbers can also be written with scientific notation to make managing large numbers easier.

For example:
var intNum = 10; // 10 without decimals
var floatNum = 10.00; // with decimals
var hugeNum = 10e5; // Returns 1000000
var x = intNum * 5; // Returns 50 in this case

Strings

Strings must be entered using quotation marks and are used to store and manipulate text. You can use either single or double quotes.

For example:
var doubleQuotes = "Hello world!" // Hello world!
var singleQuotes = 'Hello world!' // Hello world!

Booleans

Booleans are variables that return a true or false value. True means that something is correct and false means it is incorrect. A variable can be set to a boolean or a comparison can be used to return a boolean.

For example:
var isTrue = true; // True
var lessThan = 5 < 50 // True (The comparison result is correct as 5 is smaller than 50)
var isFalse = 5 > 50 // False (The comparison result is incorrect as 5 is not larger than 50)

CONFIRMATION OF KNOWLEDGE
