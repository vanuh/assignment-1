Variables

Image

Variables are containers in which we store data values. Variables allow programmers increased flexibility, as rather than entering a constant value, variables are able to change based on actions performed upon them when the program is excuted. This allows for a single program to work with different sets of data. Each variable has a name and a data type. Data types for variables include integers, floating point numbers and strings.

Datatypes

Image

JavaScript contains many different kinds of Data-types can be stored by variables. Some examples include Numbers, Strings, Objects and Lists.

Loops

Image

Loops are tools which allow us to repeat the actions of a piece of code without writing it additional times.

Conditions

Image

Conditional statements within JavaScript are indicators which tell the code to perform certain actions if a particular condition is met. Examples of these conditions include if and else statements. For example, if A is True (the condition for A is met), then B should happen. Else, C should happen.
